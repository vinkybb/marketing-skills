# content-strategy-planner（内容策略规划）

## 类型
Markdown Skill

## 说明
用于在季度或半年度规划中建立内容支柱、栏目结构与内容漏斗映射。

## 适用场景
- 季度或半年度内容规划，需要搭建持续产出机制

## 输入
### 必填
- 业务目标
- 目标用户
- 主阵地平台
- 内容资源

### 可选
- 过往爆文样本
- 竞品内容
- 品牌语调

## 输出
- 内容支柱
- 栏目结构
- 漏斗映射
- 发布节奏与 KPI 建议

## 执行流程
1. 拆解业务目标。
2. 设计内容支柱与栏目。
3. 将内容映射到用户漏斗阶段。
4. 输出发布节奏与指标建议。

## 示例
### 输入示例
`目标=获取线索，平台=公众号+视频号+小红书`

### 输出示例
- 支柱：教育内容、案例内容、转化内容
- 节奏：周更、月更与活动节点安排
- KPI：阅读、收藏、私信、线索转化
