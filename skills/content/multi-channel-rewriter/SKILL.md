# multi-channel-rewriter（多平台改写器）

## 类型
Markdown Skill

## 说明
用于将一份核心内容快速改写为适配不同平台的分发版本。

## 适用场景
- 已有核心内容，希望低成本分发到多个渠道

## 输入
### 必填
- 原始内容
- 目标平台
- 字数限制
- 语气风格

### 可选
- 禁用词
- CTA 类型
- 品牌术语

## 输出
- 平台适配稿
- 首句钩子
- CTA 建议
- 平台差异说明

## 执行流程
1. 提炼核心观点与关键信息。
2. 按平台语法改写。
3. 增加互动钩子与行动指令。
4. 统一品牌表达与 CTA。

## 示例
### 输入示例
`原文=2000 字干货，目标平台=小红书+抖音`

### 输出示例
- 小红书版：标题、分点、互动问题
- 抖音版：口播脚本、开头钩子、结尾 CTA
- 统一约束：禁用词、品牌术语
