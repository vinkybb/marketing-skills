# 模块：内容营销（Content）

## 7) content-strategy-planner（内容策略规划）
- 适用场景：季度内容规划、团队启动内容体系
- 输入：业务目标、目标用户、内容渠道、资源约束
- 输出：内容支柱、栏目规划、内容漏斗映射、发布节奏
- 执行步骤：目标拆解 → 内容支柱设计 → 漏斗映射 → 节奏排期

## 8) editorial-calendar-generator（内容日历生成）
- 适用场景：月度内容排期与跨团队协同
- 输入：月份、主题池、活动节点、平台优先级
- 输出：周度排期表、每篇内容目标、负责人建议
- 执行步骤：整理节点 → 分配主题 → 平台适配 → 形成日历表

## 9) multi-channel-rewriter（多平台改写器）
- 适用场景：一份核心内容快速适配多平台分发
- 输入：原始内容、目标平台、语气风格、字数限制
- 输出：公众号版、小红书版、知乎版、短视频口播版
- 执行步骤：抽取核心观点 → 平台语法改写 → 增加互动钩子 → 统一 CTA
