# 模块：内容营销（Content）

## 7) content-strategy-planner（内容策略规划）
- Use When：季度/半年度内容规划，需要搭建持续产出机制
- Inputs（必填）：业务目标、目标用户、主阵地平台、内容资源
- Inputs（可选）：过往爆文样本、竞品内容、品牌语调
- Outputs：内容支柱、栏目结构、漏斗映射、发布节奏与 KPI 建议
- Workflow：目标拆解 → 内容支柱设计 → 漏斗映射 → 发布节奏排期
- 示例输入：`目标=获取线索，平台=公众号+视频号+小红书`

## 8) editorial-calendar-generator（内容日历生成）
- Use When：月度执行阶段，需要把策略落到具体排期
- Inputs（必填）：月份、主题池、营销节点、平台优先级
- Inputs（可选）：团队角色、外部活动日历、预算
- Outputs：周度选题表、责任分配、发布时间建议、复盘节点
- Workflow：整理节点 → 分配主题 → 平台适配 → 输出可执行日历
- 示例输入：`月份=2026-04，节点=新品发布/五一促销`

## 9) multi-channel-rewriter（多平台改写器）
- Use When：已有核心内容，希望低成本分发到多个渠道
- Inputs（必填）：原始内容、目标平台、字数限制、语气风格
- Inputs（可选）：禁用词、CTA 类型、品牌术语
- Outputs：平台适配稿（公众号/小红书/知乎/短视频口播）
- Workflow：提炼核心观点 → 平台语法改写 → 增加互动钩子 → 统一行动指令
- 示例输入：`原文=2000 字干货，目标平台=小红书+抖音`
