# editorial-calendar-generator（内容日历生成）

## 类型
Markdown Skill

## 说明
用于把内容策略落实为可执行的月度或周度选题与排期计划。

## 适用场景
- 月度执行阶段，需要把策略落到具体排期

## 输入
### 必填
- 月份
- 主题池
- 营销节点
- 平台优先级

### 可选
- 团队角色
- 外部活动日历
- 预算

## 输出
- 周度选题表
- 责任分配
- 发布时间建议
- 复盘节点

## 执行流程
1. 整理营销节点与资源限制。
2. 给主题池分配优先级。
3. 做平台适配与节奏安排。
4. 输出带责任人的执行日历。

## 示例
### 输入示例
`月份=2026-04，节点=新品发布/五一促销`

### 输出示例
- 周历：每周主题、平台、目标动作
- 责任人：撰写、设计、审核、发布
- 复盘点：周复盘与月复盘时间
