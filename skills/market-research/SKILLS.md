# 模块：市场研究与洞察

## 1) market-landscape-scan（行业全景扫描）
- 适用场景：新项目立项、进入新赛道前的市场研判
- 输入：行业名称、地域、时间范围、关注维度（规模/增长/政策/渠道）
- 输出：行业概览、关键趋势、主要玩家、机会与风险清单
- 执行步骤：定义范围 → 收集公开资料 → 结构化归纳 → 输出机会排序

## 2) competitor-gap-analysis（竞品差距分析）
- 适用场景：需要明确与头部竞品差异，制定突破策略
- 输入：我方产品信息、3-5 个竞品、对比维度（价格/功能/渠道/内容）
- 输出：竞品对比矩阵、我方短板与可突破点、优先改进建议
- 执行步骤：建立对比维度 → 量化评分 → 找出差距 → 形成动作清单

## 3) user-voice-mining（用户声音挖掘）
- 适用场景：从评论、私信、社区讨论中提炼真实痛点
- 输入：评论文本集合、平台来源、目标用户类型
- 输出：高频痛点主题、情绪倾向、典型原话、选题建议
- 执行步骤：清洗文本 → 主题归类 → 痛点优先级排序 → 映射营销策略
