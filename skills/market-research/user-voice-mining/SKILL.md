# user-voice-mining（用户声音挖掘）

## 类型
Markdown Skill

## 说明
用于从评论、客服记录和社区讨论中提炼高频痛点、需求主题与情绪倾向。

## 适用场景
- 需要从用户评论和客服记录中提炼真实需求与抱怨点

## 输入
### 必填
- 评论文本
- 平台来源
- 目标用户类型

### 可选
- 时间范围
- 情绪标签
- 产品版本信息

## 输出
- 高频痛点主题
- 情绪分布
- 用户原话证据
- 内容与产品建议

## 执行流程
1. 清洗文本并去除噪声。
2. 聚合同类问题与主题。
3. 标记情绪与严重程度。
4. 将洞察映射到营销与产品动作。

## 示例
### 输入示例
`数据源=小红书评论+客服工单，样本量=2000 条`

### 输出示例
- 高频主题：定价、交付速度、功能门槛
- 原话摘要：典型抱怨点与购买阻力
- 行动建议：官网文案、FAQ、线索承接优化
