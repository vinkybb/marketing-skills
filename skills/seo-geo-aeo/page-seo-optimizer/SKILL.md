# page-seo-optimizer（页面 SEO 优化器）

## 类型
Markdown Skill

## 说明
用于对已有页面或即将上线页面做标题、结构、FAQ 和内链层面的优化审查。

## 适用场景
- 老页面流量下滑、新页面上线前质量检查

## 输入
### 必填
- 页面内容
- 目标关键词
- 页面目标

### 可选
- 站内链接策略
- FAQ 池
- 品牌术语

## 输出
- 标题与描述建议
- H 结构建议
- FAQ 与内链建议
- 修复清单

## 执行流程
1. 审查结构与信息层次。
2. 检查关键词布局。
3. 优化可读性与转化承接。
4. 输出修复优先级清单。

## 示例
### 输入示例
`页面=产品页，关键词=线索培育系统`

### 输出示例
- 标题建议：聚焦主词与收益
- 结构建议：H1/H2/H3 调整
- 补充项：FAQ、内链、信任证据
