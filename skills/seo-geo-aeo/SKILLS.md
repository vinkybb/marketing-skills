# 模块：SEO / GEO / AEO

## 10) keyword-cluster-builder（关键词聚类）
- 适用场景：搭建 SEO 内容矩阵与站点主题结构
- 输入：种子关键词、业务领域、地域/语言
- 输出：关键词簇、主词-长尾词映射、优先级清单
- 执行步骤：扩词 → 清理噪声 → 意图分组 → 难度与价值排序

## 11) serp-intent-analyzer（SERP 意图分析）
- 适用场景：写作前判断用户搜索背后真实需求
- 输入：关键词列表、目标搜索引擎、竞品页面
- 输出：搜索意图标签、内容形式建议、结构建议
- 执行步骤：采样结果页 → 识别意图类型 → 归纳内容模式 → 输出建议

## 12) page-seo-optimizer（页面 SEO 优化器）
- 适用场景：现有页面优化与排名修复
- 输入：页面草稿或已发布内容、目标关键词、页面目标
- 输出：标题建议、H 结构建议、FAQ 建议、内链建议
- 执行步骤：结构审查 → 关键词布局 → 可读性优化 → 技术检查清单

## 13) answer-engine-optimizer（AEO 答案引擎优化）
- 适用场景：提升在 AI 问答结果中的引用概率
- 输入：目标问题集合、现有内容、品牌主张
- 输出：问答对内容块、可引用段落、来源可信度增强建议
- 执行步骤：问题标准化 → 生成高密度答案块 → 补充证据结构 → 统一语义
