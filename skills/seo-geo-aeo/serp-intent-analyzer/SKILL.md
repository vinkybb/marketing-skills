# serp-intent-analyzer（SERP 意图分析）

## 类型
Markdown Skill

## 说明
用于在写内容前判断搜索意图、头部结果结构和更合适的内容形式。

## 适用场景
- 写内容前先判断搜索意图和结果页竞争结构

## 输入
### 必填
- 关键词列表
- 目标搜索引擎

### 可选
- 竞品 URL
- 目标排名区间

## 输出
- 意图标签
- 内容类型建议
- 页面结构建议
- 头部结果共性总结

## 执行流程
1. 采样结果页。
2. 分类搜索意图。
3. 总结头部页面结构和内容形式。
4. 输出页面与内容建议。

## 示例
### 输入示例
`关键词=营销自动化工具推荐`

### 输出示例
- 意图：商业调查
- 推荐内容：对比评测、选型指南、FAQ
- 页面结构：榜单、评测维度、购买建议
