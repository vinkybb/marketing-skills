# 模块：CRM 与私域运营

## 19) lifecycle-segmentation（生命周期分层）
- 适用场景：建立精细化运营，避免“一刀切”触达
- 输入：用户行为数据、交易记录、最近活跃时间
- 输出：新客/活跃/沉睡/高价值等分层规则与运营动作
- 执行步骤：定义分层规则 → 人群打标 → 策略映射 → 触达优先级

## 20) winback-automation-planner（召回自动化流程）
- 适用场景：沉睡用户召回、老客复购提升
- 输入：目标人群、可用触达渠道、权益资源、触达频次限制
- 输出：多触点召回流程、消息模板、失败分支策略
- 执行步骤：召回分群 → 流程编排 → 内容模板生成 → 监控指标设置
