# 模块：CRM 与私域运营（CRM & Private Domain）

## 19) lifecycle-segmentation（生命周期分层）
- Use When：用户触达粗放、复购不稳定、留存下滑
- Inputs（必填）：用户行为数据、交易记录、最近活跃时间
- Inputs（可选）：会员等级、渠道来源、历史优惠使用记录
- Outputs：生命周期分层规则、分层运营动作、触达频率建议
- Workflow：定义分层标准 → 人群打标 → 运营动作映射 → 形成执行优先级
- 示例输入：`分层目标=新客/活跃/沉睡/高价值`

## 20) winback-automation-planner（召回自动化流程）
- Use When：沉睡用户增长、复购率下降、召回流程需要标准化
- Inputs（必填）：目标人群、可用渠道、权益资源、触达频控规则
- Inputs（可选）：历史召回成功样本、渠道成本、客服承接能力
- Outputs：召回流程图、消息模板、分支策略、监控指标
- Workflow：召回分群 → 触达编排 → 内容模板生成 → 指标监控设计
- 示例输入：`渠道=短信+企微+公众号，频控=7 天不超过 3 次`
