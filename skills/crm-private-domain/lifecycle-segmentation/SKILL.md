# lifecycle-segmentation（生命周期分层）

## 类型
Markdown Skill

## 说明
用于根据用户行为与交易记录建立分层规则，支撑更精细的运营触达。

## 适用场景
- 用户触达粗放、复购不稳定、留存下滑

## 输入
### 必填
- 用户行为数据
- 交易记录
- 最近活跃时间

### 可选
- 会员等级
- 渠道来源
- 历史优惠使用记录

## 输出
- 生命周期分层规则
- 分层运营动作
- 触达频率建议

## 执行流程
1. 定义分层标准。
2. 完成人群打标。
3. 将人群映射到运营动作。
4. 输出优先级与频控建议。

## 示例
### 输入示例
`分层目标=新客/活跃/沉睡/高价值`

### 输出示例
- 分层标准：时间、行为、价值维度
- 运营动作：教育、促活、召回、会员权益
- 频控：不同层级的触达上限
