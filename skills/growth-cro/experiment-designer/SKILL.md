# experiment-designer（增长实验设计）

## 类型
Markdown Skill

## 说明
用于将优化建议转成可验证的增长实验，包括假设、分组设计和成功标准。

## 适用场景
- 需要把优化建议转为可验证的增长实验

## 输入
### 必填
- 实验目标
- 基线指标
- 可改动变量
- 实验周期

### 可选
- 风险约束
- 流量上限
- 工程依赖

## 输出
- 实验假设
- 分组设计
- 成功判定标准
- 复盘框架

## 执行流程
1. 提出可验证假设。
2. 设计对照与实验组。
3. 明确观测指标与阈值。
4. 预设复盘框架与失败处理方案。

## 示例
### 输入示例
`目标=注册转化+15%，变量=表单长度/按钮文案`

### 输出示例
- 假设：缩短表单字段提升转化
- 分组：控制组与两组实验组
- 判定：主要指标、次要指标与终止条件
