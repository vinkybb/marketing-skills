# 模块：增长与转化（Growth / CRO）

## 17) funnel-diagnostics（增长漏斗诊断）
- 适用场景：转化率停滞、各环节效率不清晰
- 输入：曝光/点击/注册/激活/付费等漏斗数据
- 输出：瓶颈环节定位、可能原因、优先修复建议
- 执行步骤：分层看漏斗 → 识别异常段 → 假设归因 → 输出行动项

## 18) experiment-designer（增长实验设计）
- 适用场景：需要将策略建议转为可执行实验
- 输入：实验目标、当前基线、可改动元素、实验周期
- 输出：实验假设、变量定义、样本建议、结果判定规则
- 执行步骤：提出假设 → 设计对照组 → 定义指标 → 预设复盘模板

## 23) ab-test-sample-size（A/B 样本量计算器）
- 适用场景：上线实验前估算每组最小样本量
- 输入：baseline_rate、mde、alpha、power
- 输出：每组样本量、总样本量、参数回显
- 执行步骤：读取参数 → 计算双侧检验样本量 → 输出结构化结果
- Python 实现：`skills/growth-cro/ab_test_sample_size.py`
