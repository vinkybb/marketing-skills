# marketing-kpi-framework（营销指标体系搭建）

## 类型
Markdown Skill

## 说明
用于在跨团队协作中统一指标口径、层级与预警机制，避免只看表面数据。

## 适用场景
- 指标口径不统一、团队只看表面数据、难以驱动决策

## 输入
### 必填
- 业务目标
- 数据字段
- 现有报表结构

### 可选
- 组织角色
- 指标刷新频率
- 阈值偏好

## 输出
- 北极星指标
- 分层指标树
- 统一口径说明
- 预警阈值

## 执行流程
1. 对齐业务目标。
2. 设计指标分层与归因关系。
3. 明确统计口径。
4. 输出预警和复盘机制。

## 示例
### 输入示例
`目标=新增付费用户，周期=季度，渠道=全渠道`

### 输出示例
- 北极星指标：新增付费用户数
- 支撑指标：线索、转化、留存、成本
- 预警规则：异常阈值与责任归属
