# 模块：数据分析与归因

## 21) marketing-kpi-framework（营销指标体系搭建）
- 适用场景：团队指标混乱、跨部门口径不一致
- 输入：业务目标、渠道数据字段、当前报表结构
- 输出：北极星指标、分层指标树、口径说明与阈值建议
- 执行步骤：目标对齐 → 指标分层 → 口径统一 → 预警阈值定义

## 22) campaign-retrospective-writer（活动复盘生成）
- 适用场景：活动结束后快速形成复盘结论与下轮动作
- 输入：活动目标、执行数据、素材与渠道表现、异常记录
- 输出：结果摘要、成功因素、问题清单、下周期优化建议
- 执行步骤：目标回看 → 数据对照 → 问题归因 → 策略建议输出
