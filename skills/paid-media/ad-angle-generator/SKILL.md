# ad-angle-generator（广告创意角度生成）

## 类型
Markdown Skill

## 说明
用于围绕产品卖点与受众动机快速扩展广告创意角度和标题方向。

## 适用场景
- CTR 下滑、素材疲劳、需要快速扩展创意池

## 输入
### 必填
- 产品卖点
- 目标人群
- 投放平台

### 可选
- 禁用表达
- 品牌语调
- 竞品广告样本

## 输出
- 利益型角度
- 痛点型角度
- 对比型角度
- 场景型角度与标题方向

## 执行流程
1. 拆解卖点与受众动机。
2. 映射不同创意结构。
3. 结合平台语言生成角度。
4. 做首轮筛选与风险检查。

## 示例
### 输入示例
`卖点=7 天见效，受众=中小企业主，平台=抖音`

### 输出示例
- 痛点型：解决获客慢的问题
- 场景型：老板和团队日常低效的对比
- 对比型：传统方案与新方案效果差异
