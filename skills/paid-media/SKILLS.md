# 模块：广告投放与创意优化（Paid Media）

## 14) channel-budget-allocator（渠道预算分配）
- 适用场景：新季度预算规划与投放组合设计
- 输入：总预算、目标（线索/成交/安装）、渠道候选、历史数据
- 输出：渠道预算建议、测试预算比例、保守与激进两套方案
- 执行步骤：目标优先级定义 → 渠道能力评估 → 预算切分 → 风险缓冲

## 15) ad-angle-generator（广告创意角度生成）
- 适用场景：创意疲劳、点击率下降时快速扩展素材方向
- 输入：产品卖点、目标人群、平台、禁用词要求
- 输出：利益型/痛点型/对比型/故事型创意角度与标题
- 执行步骤：卖点拆解 → 角度映射 → 平台语境改写 → 首轮筛选

## 16) ad-copy-variant-generator（广告文案变体生成）
- 适用场景：同一创意做 A/B 测试，快速迭代文案
- 输入：广告角度、长度限制、CTA 类型、平台规范
- 输出：短文案、中长文案、首句钩子、CTA 变体
- 执行步骤：结构模板选择 → 变体生成 → 合规检查 → 测试分组输出
