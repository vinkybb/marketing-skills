# ad-copy-variant-generator（广告文案变体生成）

## 类型
Markdown Skill

## 说明
用于在同一创意方向下快速输出多组广告文案，支持 A/B 测试与平台适配。

## 适用场景
- 同一素材需要批量 A/B 测试不同文案结构

## 输入
### 必填
- 创意角度
- 字数限制
- CTA 类型
- 平台规范

### 可选
- 落地页承接点
- 优惠信息
- 品牌名出现频次

## 输出
- 短文案
- 中长文案
- 首句钩子
- CTA 变体与测试分组建议

## 执行流程
1. 选择文案结构模板。
2. 生成多组变体。
3. 做平台合规检查。
4. 输出测试分组方案。

## 示例
### 输入示例
`角度=痛点型，CTA=立即咨询，字数<45`

### 输出示例
- 变体 A：直接痛点 + CTA
- 变体 B：对比收益 + CTA
- 测试建议：高承诺版与低承诺版分组
