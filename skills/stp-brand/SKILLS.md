# 模块：STP 与品牌定位

## 4) icp-builder（ICP 客户画像构建）
- 适用场景：B2B/B2C 营销前明确理想客户范围
- 输入：历史高价值客户样本、客单价、行业/岗位/场景信息
- 输出：ICP 描述、排除人群、触达渠道建议
- 执行步骤：识别高价值样本 → 提炼共同特征 → 形成画像模板

## 5) persona-generator（Persona 角色画像生成）
- 适用场景：内容团队与投放团队统一用户认知
- 输入：目标人群、核心需求、购买阻力、决策链角色
- 输出：3-5 类 Persona 卡片（目标、痛点、触发、阻碍、偏好）
- 执行步骤：人群分群 → 角色命名 → 需求与阻碍建模 → 触点映射

## 6) positioning-statement-crafter（品牌定位声明生成）
- 适用场景：品牌升级、官网改版、对外传播统一话术
- 输入：目标人群、品类、核心收益、差异化证据
- 输出：定位声明、价值主张、辅助卖点、禁用表达
- 执行步骤：明确竞争语境 → 提炼差异点 → 组合定位句 → 校验一致性
