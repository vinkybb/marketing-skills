# 模块：STP 与品牌定位（STP & Brand）

## 4) icp-builder（ICP 客户画像构建）
- Use When：销售线索质量不稳定、市场投放人群过宽、需要明确高价值人群
- Inputs（必填）：高价值客户样本、行业/角色信息、客单价或生命周期价值
- Inputs（可选）：线索来源、成交周期、续费率
- Outputs：ICP 主画像、排除画像、优先触达渠道、关键信息点
- Workflow：筛选高价值样本 → 提炼共性特征 → 构建 ICP 模板 → 给出渠道建议
- 示例输入：`样本=近 6 个月成交客户，行业=SaaS，客单价>=2 万`

## 5) persona-generator（Persona 角色画像生成）
- Use When：内容、投放、销售团队对目标用户理解不一致
- Inputs（必填）：目标人群、核心任务、购买阻力、决策链角色
- Inputs（可选）：使用场景、预算约束、偏好平台
- Outputs：3-5 个 Persona 卡片（目标/痛点/触发/阻碍/内容偏好）
- Workflow：细分人群 → 构建角色卡 → 映射触点 → 形成沟通建议
- 示例输入：`角色=市场经理/老板/运营，目标=提升留资转化`

## 6) positioning-statement-crafter（品牌定位声明生成）
- Use When：品牌升级、官网改版、融资路演、对外口径统一
- Inputs（必填）：目标客群、品类、核心收益、差异化证据
- Inputs（可选）：品牌语调、竞争对手定位、合规限制
- Outputs：定位声明、价值主张、支撑卖点、禁用词清单
- Workflow：定义竞争语境 → 提炼差异化 → 组合定位语句 → 一致性审校
- 示例输入：`品类=营销自动化，核心收益=提升线索到成交效率`
