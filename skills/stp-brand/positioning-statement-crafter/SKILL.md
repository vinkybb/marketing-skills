# positioning-statement-crafter（品牌定位声明生成）

## 类型
Markdown Skill

## 说明
用于统一品牌对外口径，沉淀定位声明、价值主张与差异化表达。

## 适用场景
- 品牌升级、官网改版、融资路演、对外口径统一

## 输入
### 必填
- 目标客群
- 品类
- 核心收益
- 差异化证据

### 可选
- 品牌语调
- 竞争对手定位
- 合规限制

## 输出
- 定位声明
- 价值主张
- 支撑卖点
- 禁用词清单

## 执行流程
1. 定义竞争语境。
2. 提炼差异化证据。
3. 组合定位语句与支撑卖点。
4. 做一致性与合规审校。

## 示例
### 输入示例
`品类=营销自动化，核心收益=提升线索到成交效率`

### 输出示例
- 定位句：面向特定客户与场景的核心价值表达
- 支撑点：效率、成本、效果、证据来源
- 禁用表达：容易产生误导的宽泛承诺
