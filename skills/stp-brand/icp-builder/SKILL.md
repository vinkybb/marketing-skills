# icp-builder（ICP 客户画像构建）

## 类型
Markdown Skill

## 说明
用于从高价值客户样本中提炼理想客户画像，减少人群过宽带来的投放浪费。

## 适用场景
- 销售线索质量不稳定、市场投放人群过宽、需要明确高价值人群

## 输入
### 必填
- 高价值客户样本
- 行业或角色信息
- 客单价或生命周期价值

### 可选
- 线索来源
- 成交周期
- 续费率

## 输出
- ICP 主画像
- 排除画像
- 优先触达渠道
- 关键信息点

## 执行流程
1. 筛选高价值样本。
2. 提炼共同属性与购买条件。
3. 形成主画像与排除画像。
4. 输出渠道与沟通建议。

## 示例
### 输入示例
`样本=近 6 个月成交客户，行业=SaaS，客单价>=2 万`

### 输出示例
- ICP：角色、团队规模、预算能力、核心任务
- 排除条件：低预算、短周期不匹配场景
- 建议渠道：内容、广告、销售外呼优先级
